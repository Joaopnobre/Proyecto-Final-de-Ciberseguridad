# Proyecto-Final-de-Ciberseguridad
Proyecto Final de Ciberseguridad de 4Geeks Academy

Proyecto práctico de análisis, identificación y corrección de vulnerabilidades en un servidor Debian Linux utilizando Kali Linux en un entorno controlado.

---

## Objetivos del Proyecto

- Identificar vulnerabilidades en servicios expuestos
- Realizar pruebas de explotación controlada
- Aplicar medidas correctivas
- Validar las correcciones implementadas
- Documentar incidentes y recomendaciones de seguridad
- Implementar controles básicos alineados con ISO 27001
